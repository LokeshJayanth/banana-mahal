# Images Folder

Place your converted WebP images here:

- hero.webp (Main hero banner - recommended 1920x1080px)
- hall1.webp (Gallery image 1)
- hall2.webp (Gallery image 2)
- hall3.webp (Gallery image 3)
- logo.webp (Optional logo)

## Convert Your Images to WebP

Use online tools like:
- https://squoosh.app/
- https://cloudconvert.com/jpg-to-webp

Or use the img folder images you already have and convert them to WebP format.
